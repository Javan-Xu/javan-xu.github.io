The Power of Unrecognized

Posted on 2021-11-02 Edited on 2021-11-06
Symbols count in article: 1.1k Reading time ≈ 2 mins.

自從九月底離開前公司後，持續地修改著履歷資料，也陸續地投出了幾份履歷，扣除那些單純在打廣告 (連投遞開啟通知都沒有) 的職缺，難免還是會有被拒絕的狀況，好一點的公司會回覆個婉謝函，但其它大多數都是看了之後就沒下聞了。問說心理不會受到打擊? 那是難免的。

供給與需求

其實會有這樣的狀況我並不意外，因為為了要找到自己覺得較好或較適合的職缺，所以我 104 的履歷一直都是關閉、主動投遞的狀態。公司挑員工跟我們挑公司，本來就是個供需的體現，當你用高的標準去過濾你覺得適合的職缺，當然對方也會用高的標準來過濾適合他們職務的人員，這本來就是個很正常、很公平的狀況。

另一個我比較擔心的是 “年紀” ，我現在也已經四十初頭歲了，如果企業是因為這個因素而不敢用我，那也怪不得他們，只是這樣子真的很無奈啊! 畢竟這又不是努力就可以改變的事情。只能奉勸要轉職的大家，如果您這時候已經超過四十歲的話，真的要做好評估後再行動。

蓄勢待發

過了一個月、下一份工作卻還沒有下聞，這得確對我家庭經濟跟心理上有很大的壓力。現在回想起來，我還是覺得離開是有必要的，因為雖然我從出社會工作到現在都持續在學習新的技術，也一直想把英文學好，但是最近這一年多的時間，是我感覺到這股前進的力量最明顯也最強烈的時候。

以英文的學習來說，雖然說英文還是一樣很爛，但是因為主管要求每天都要寫工作日誌，剛好公司用到英文的機會也多，我從一開始就用英文寫工作日誌，就這樣持續了一年多，不敢說英文就變得多好，但是愈到後頭、文法檢查工具出現的錯誤數量得確變少了很多。

在技術面，那些在開發設備控制系統從無到有之間要學習、應用的技術跟套件當然都是新的收獲，但最讓我有感的是用 Python coding 的速度，這比我之前用過的其它程式語言快上了許多。我自己綜合歸納起來，有可能是 Python 本身語法的特性，跟我短時間內 coding 的頻率，也有可能是 IDE PyCharm 好用的快捷鍵跟外掛工具，也有可能是一直都在 Mac 上進行開發的關係。

但是這些我覺得我正快速成長的 coding、英文跟新事物學習應用能力，在前公司卻不是會受到重視及鼓勵的事情，相反的，它還可能會受到一些限制。所以我才會說，離開是有必要的。

只是現在處於這轉職的關卡，沒辦法如願地應徵上理想中的職缺，對照著那股蓄勢待發的感覺跟持續被否定的次數，內心真的是五味雜陳啊! 也只能安慰自已說 “隨緣” 了，期待看看會不會有個不錯的職缺跟公司能讓我接續這份成長的動能了，屆時我一定會牢記這份被否定的感覺，為那個願意接受我這個中年男人的公司多盡一份心力的。

關於開發成果

Posted on 2021-09-14 Edited on 2021-09-23
Symbols count in article: 2.2k Reading time ≈ 4 mins.

以共好來代替藏私

還記得 2014 年在中興大學圖書館工作的時候，因為在管理圖書館自動化系統上做了很多的應用，受邀去跟大家做了一場「系統管理及加值應用經驗分享」。簡報的最後我跟大家分享了這兩句話。

眼光要放長遠一點，「以擁有知識、技能來確保自己存在的價值」是等著被時代淘汰的舊思維，因為我們無法去限制他人的成長。

「如何透過跟他人的分享合作來創造自身差異化的價值」，這才是未來發展的王道。

那時候之所以會有這樣子的想法，就要來說點當時的一些小故事。

因為當時國內大學相當注重在西班牙世界網路大學的排名(其實就我來看是很沒有意義)，而圖書館因為負責校內機構典藏系統、裡頭收納了很多校內學術發表資料，而這些學術資料會直接影響到該網路排名裡頭相當重要的 Scholar 項目分數，所以當時圖書館對於如何提升學校在世界網路大學排名這件事情上影響很大，當然也倍感壓力，因為學校高層也一直在盯這件事情。

而小弟我，剛好是負責管理這些圖書管資訊系統，又好死不死地，被我試出來只要提高機構典藏系統在 Google Analytics 的索引速度，就可以讓該網站的學術資料在極短時間內出現在搜尋引撆上，就能使得西班牙世界網路大學排名中的 Scholar 項目分數迅速提升。(當然啦! 這個目前已經不是秘密了，後來官方似乎也有查覺這個漏洞，在評分權重跟方法有做了修改。)

也就因為這樣子，我們圖書館就讓中興大學在西班牙世界網路大學排名的 Scholar 項目由 2009 年 7 月的 804 名前進至 2010 年 1 月的全球第 3 名，連帶的，整體的排名也一口氣跑到了很前面(但詳細已經忘記是幾名了)。

那其它國內圖書館當然也看到了這個狀況，紛紛跑來詢問我們是怎麼做的。
當時我們館內的其他同事就是裝傻、想保密，但我自己是比較傾向跟其它學校分享的，因為我認為，這種東西就算我們不講，別人自己試個一年半載也終究會發現的。而且，畢竟這是個世界網路大學排名，我們國內學校應該要團結一起提升在世界上的學校能見度才對，怎麼會這麼小鼻子小眼睛在防自己人呢!?

所以從那時候我就覺得對於開發成果這件事情，不應該是狹隘地隱藏來確保自身的競爭優勢，而是應該跟大家一起共好才對。我們這些國內的圖書館應該要互相交流、分享應用成果，然後各自發展每個學校差異化的特點，而不是互相防來防去的，然後再來一直羨慕國外圖書館做得有多好。

這個觀念如果是套在一般私人公司也是相同的道理，同一個公司內的部門之間，同一個部門的同事之間，不應該是相互藏私的，應該是相互分享協助來讓組織的競爭力極大化，因為我們的敵人是其它公司，甚至是國外的廠商，眼光應該要放遠長遠一點。

要擔心的應該是自己停滯不前，而不是別人的模仿

在離開中興大學圖書館後，我到營造公司的 BIM(數位資訊模型)部門服務，主要是在做 BIM、IoT 跟職業安全相關的研發工作，理所當然地，我的工作成果就時常會是部門在公司工作報告時的”亮點”，甚至是公司在新案投標、參獎 (金安獎、金質獎) 或是對外宣傳時的”亮點”。

也就這樣，每年為部門跟公司構思並完成一些上得了檯面的應用成果就成了我的工作，每次有外賓參訪、公司參獎，要介紹 BIM 應用成果時我就會被推出去做介紹，每年也都會被叫去台大舉辦的 BIM 年度應用成果展介紹公司這一年多做了哪些創新應用。

我知道那時候參加這類的成果介紹活動，公司高層是有點反對的，因為他們會覺得在那個場合把自己的一些開發成果展示出來，可能會讓其它同業學走而讓公司失去競爭優勢。但我對這個是有不同看法的。

  • 第一個是，就如同我前頭提到的，我希望這會是個共好的狀態，大家一起相互觀摩學習，才能讓這個產業更加進步。(好在我們主管的樣法跟我一樣)
  • 第二個是，我壓根就不覺得所謂的競爭優勢是這些已經做出來的成果，所謂的競爭優勢應該是持續創新的能力才對。其它同業要學又怎樣，當它做出來相同的東西時，我們早就又已經提出其它的創新做法了。我們要擔心的應該是自己停滯不前，而不是擔心別人的模仿。

最有價值的，其實是完成那些東西的人，而不是那些程式碼

最近，因為一些工作理念跟環境因素，準備要來更換工作。
每每在這個工作交接的時期，公司都會讓我有種「深怕我們會藏私、不會把工作成果完全交接給其他同事」的感覺。其實了解我的人就知道，我是不會做這種事情的，因為我領公司薪水開發出來的成果，本來就應該交接給公司。
而且講白了，那些開發成果根本就不是什麼重要的東西，它們充其量只是一堆程式碼而已，真正重要、有價值的，是想出這個系統架構、完成這些整合應用、了解這些應用重點，並把它實現出來的那個人。

沒錯，或許公司很有錢，不怕請不到人，但是一個人員從進到公司; 熟悉業務; 到可以做出高質量的貢獻是需要時間的，這依照人員素質，通常需要半年到一兩年的時間。
如果人員的流動率一直很高的話，那麼這個公司或部門隱形的營運成本絕對會很高，實在很難讓人能相信它能提供多麼穩定且高品質的產品或服務。
一個真的在乎這個問題的人，應該找出、面對並解決導致人員高流動率的問題才對。

隨緣，不用於太過執著

就像我跟交接工作的新同事說的，每個公司、主管跟人員都會有各個面向，也無所謂一定的好或壞，就端看彼此間是否能相互認同罷了。
或許我們很不能忍受的人事物，在他人眼裡是很 ok 的，所以我們也沒必要把自己的價值觀強加到別人身上。

總歸一句話，就是 “隨緣” 啦!

Self-Reflection

Posted on 2021-09-07 Edited on 2021-11-23
Symbols count in article: 502 Reading time ≈ 1 mins.

這是一些我自己工作、生活上持續記錄下來的一些自我檢討跟提醒的事，內容會持續更新 ，也跟有緣的你一起分享。

態度

  • 別只要求別人給我什麼，想想自己為別人做了什麼?
  • 他人的抱怨與閒語，代表著自己做得不夠好、不夠完善。
  • 別去責怪別人不相信你，應該要檢討為什麼別人不相信你。
  • 不要因為沒有掌聲，而放棄了自己該堅持的理; 不要因為沒有雨露，而忘記了自己該努力的成長。
  • 人的一生中有很多被選擇的無奈。因此我們要珍惜少數可以選擇的機會，創造更多可以選擇的條件。

溝通服務

  • 生氣時，先數到 10 再說話; 如果還在氣，那就數到 100。
  • 生氣時，先想想你是要吐一口氣? 還是要解決問題?
  • 如果有人老是找你麻煩，你要原諒他，因為他一定很不快樂。
  • 溝通最重要的是去了解對方 ”沒有說出來” 的部分。
  • 溝通，不是為了改變對方，而是找到彼此都能理解的地方。

做事

  • Timing 很重要，做事要把握時機。
  • 可以隨手處理掉的事，要馬上立即處理。
  • 今天能做完的事情絕不拖到明天。
  • 件件工作反映自我，凡經我手必為佳作。
  • 有力能的人影響別人，沒能力的人受人影響。
  • 君子，不自大其事，不自尚其功，不掠人之美。
    由他人口中說出的你的好，才是真正的好。
  • The more we do, the more we can do.
